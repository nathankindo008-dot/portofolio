Prédiction COVID-19 - Classification Médicale

Projet de classification binaire réalisé via Machine Learnia pour prédire si un patient est atteint du COVID-19 ou non. Machine Learning Classification Medical AI Data Analysis Visualization Tests Statistiques Optimization
