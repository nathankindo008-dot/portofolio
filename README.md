# Nathan Kindo Portfolio

Portfolio personnel en ligne de **Nathan Kindo Pawentaore** (Analyste Statisticien / Data Scientist), construit en HTML, CSS et JavaScript avec version bilingue FR/EN.

## Apercu

- Site vitrine professionnel avec sections : A propos, Parcours, Projets, Publications, Contact.
- Projets Data Science / IA / Econometrie avec stack technique visible.
- Internationalisation FR/EN avec bascule dynamique.
- Integration d'assets (certificats, rapports PDF, image de dashboard Power BI).

## Stack

- `HTML5`
- `CSS3`
- `JavaScript (Vanilla)`
- `GitHub Pages` (deploiement)

## Projets mis en avant

- Comparateur de Prix - Jumia CI (pipeline scraping, API, orchestration)
- Dashboard Power BI - Analyse Coupe du Monde
- Deep Learning - Detection du Cancer du Sein
- Prediction COVID-19 - Classification Medicale

## Lancer en local

Comme le projet est statique, ouvre simplement `index.html` dans un navigateur.

Option recommandee (VS Code / Cursor) :

1. Installer l'extension Live Server.
2. Lancer "Open with Live Server" sur `index.html`.

## Deploiement

Le site est publie via GitHub Pages depuis la branche `main`.
